## Getting Started

Lenguaje Dart flutter, Nivel principiante, Ejemplo de navegacion entre 3 pagina utilizando rutas nombradas, desde main llamar a pagina 1, En la pagina 1mostrar en appbar el titulo "inicio" color blanco fondo negro y en body y en una columana mostrar texto (pagina inicial)  mas un boton azul marino o fuerte para ir a la segunda pagina,  en la segunda pagina mostrar el appbar "segund pagina" color blanco con fondo gris dagradado y en body mostrar una imagen  desde la red mas un boton  azul como gris para ir a la tercera pagina, en la tercera pagina en appbar mostrar  titulo "Tercera pagina" color blanco y gris degradado  y un body mostrar un conainer de 200 gris claro, debe ser actrativo facil navegacion y mostrar todo el codigo en un solo archivo
